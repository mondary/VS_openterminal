## ADDED Requirements
### Requirement: Editor Area Terminal Button
The extension SHALL contribute a toolbar button in the editor area that opens a new terminal in a new editor tab, positioned to the left of "Split Editor Right".

#### Scenario: User clicks editor-area button
- **WHEN** the user clicks the editor-area button
- **THEN** a new terminal opens and is shown in the editor area as a tab

### Requirement: Command Palette Access
The extension SHALL provide a command that opens a new terminal in the editor area.

#### Scenario: User runs the command
- **WHEN** the user runs the command from the Command Palette
- **THEN** a new terminal opens and is shown in the editor area as a tab
